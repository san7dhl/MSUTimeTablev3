User Manual

1

Setup & Data Import

Begin by populating your workspace. Go to the Data & Settings tab. The quickest way to start is by downloading the Excel template, filling it with your university's data (Programs, Faculty, Courses), and uploading it using the "Import Data" button. Alternatively, you can add data manually using the "Individual" or "Bulk Edit" modes.

2

Scheduling Classes

Navigate to the Student Group Schedule tab. Use the dropdowns on the left to select a Department, Program, and Semester. Courses for that group will appear below. Simply drag a course from the list and drop it onto an available time slot in the grid.

3

Viewing & Verification

Switch between views using the top tabs. The Faculty Schedule tab is perfect for verifying that no faculty member is overbooked. The tool will automatically show workload warnings and prevent direct time-slot clashes.

4

Collaboration & Exporting

To work with colleagues, click the "Share Workspace" button and send them the URL. All changes will sync in real-time. Once your timetables are complete, use the "Export" buttons to download formatted Excel files for distribution or printing.

Pro-Tips

  • Parent Programs: Use the "Parent Programs" field when creating or editing a program to automatically share common courses between different student groups.
  • Data Health Check: Periodically run the "Health Check" tool in Data & Settings to find and fix inconsistencies like unassigned courses or potential typos.
  • Clear Schedule: If you need to restart a specific group's timetable, the "Clear" button on the grid view will only remove entries for that group, leaving parent schedules intact.
